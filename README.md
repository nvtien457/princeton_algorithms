## Fix bugs

### **```command not found javac-algs4```**
* Create `javac-algs4.sh` follows https://algs4.cs.princeton.edu/mac/javac-algs4 
* Edit ```INSTALL=/usr/local/algs4``` with custom folder containing `algs.jar`.
* Run command `javac-algs4 <filename>.java` with: ```bash javac-algs4.sh <filename>.java```

### **```command not found java-algs4```**
* Create `java-algs4.sh` follows https://algs4.cs.princeton.edu/mac/java-algs4
* Edit ```INSTALL=/usr/local/algs4``` with custom folder containing `algs.jar`.
* Run command ```java-algs4 <filename>``` with: ```bash java-algs4.sh <filename>```
